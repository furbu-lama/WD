What is Hypertext Transfer Protocol (HTTP)?

The ypertext Transfer Protocol (HTTP) is application protocol for distributed, collaborative, hypermedia information system where hypertext document include hyperlink to other resources which can easily be assessed.

Why is HTTP called a stateless protocol?

Every action that server takes or response code may be different where methods like GET, HEAD, OPTIONS are being prescribed as safe, should need to be idempotent.That's why HTTP is a stateless protocol.

Describe the three phases of establishing a session using client-server protocols such as HTTP

It consist of three phases like HTTP:

  • The client establish a TCP connection for appropriate connection if transport layer is not TCP.
  • The client send request and wait for reply.
  • The server process the request providing status code and appropriate answer.

    • What are the HTTP response status code and What do the HTTP status codes, 200, and 404 indicate?

    HTTP response ststus code are seprated into 5 categories

  • 1xx (Informational): The request was received, continuing process
  • 2xx (Successful): The request was successfully received, understood, and accepted
  • 3xx (Redirection): Further action needs to be taken in order to complete the request
  • 4xx (Client Error): The request contains bad syntax or cannot be fulfilled
  • 5xx (Server Error): The server failed to fulfill an apparently valid request
      • HTTP 404 indicate Not Found, Page Not Found, Server Not Found
      HTTP 200 indicate successful responses, received, understood and accepted.
